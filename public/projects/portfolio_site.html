Portfolio Website

Contact Form

The contact form is generated via Javascript functions when the Contact Me button is clicked. While it would be better from a performance perspective to have a pre-built form that is hidden or revealed, this made for a good exercise in document fragments and dynamic node selection.

API Integration

When the contact form is submitted, it conatcts an endpoint running on a Node.js server. That API translates the form details to the Resend API, which sends an email containing the sender's details and message. A dedicated inbox at samuel@samueldaley.com is where the messages are sent to.

Responsiveness

Media Queries are present for a few breakpoints to allow the site to adjust gracefully to any screen size. The CSS grid allowed the layout to easily change to new arrangements as needed.
